NÁPOVEDA

Pre prvé štyri kanály (ta3, joj, joj+, wau) je potrebné mať nainštalovaný flash plugin (čo má v dnešnej dobe vačšina ľudí).
Ak Vám ide Youtube tak ho máte z najvačšou pravdepodobnosťou nainštalovaný.

Pre všetky ostatné kanály je potrebný extra plugin pre prehliadač.

Mozilla Firefox

Tí čo používate prehliadač Mozilla Firefox je potrebné nainštalovať VLC prehrávač.

Pre pozeranie odporúčam používať Mozillu Firefox + VLC. Dekódvanie videa ide rýchlejšie a úplne plynule, v Chrome to dekódovanie trochu seká.

Chrome

Tí čo používate Chrome je potrebné nainštalovať VXG Plugin pre chrome a stránka by Vás mala na to vyzvať a stačí len preklikať inštaláciu.

TV kanály

Pre pozeranie televíznych kanálov je potrebné mať nainštalovanú aplikáciu:
pre iPhone:VLC pre android pre Android:VLC pre iPhone

Markíza / JOJ Archív

1. Kliknite na menu v ľavom hornom rohu a zvoľte reláciu z JOJ archívu

2. Zvoľte želanú epizódu zo zoznamu

3. Po načítaní videa kliknite tlačítko prehrávať
